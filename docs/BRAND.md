# SailBook — Brand Direction

## Name
SailBook

## Tagline
Scheduling for Learn To Sail Cleveland (internal tool — no public marketing needed for V1)

## Voice
Practical, clean, no-nonsense. This is a tool for running a sailing school, not a consumer app. Admin screens should feel like a well-organized dashboard. Student screens should feel simple and clear — browse, register, done.

## Visual Direction
- Oleg's Law: you can use any color you want as long as it's white or black. Navy and gray are the accent palette.
- Sans-serif typography only. Clean above all else.
- No decoration for decoration's sake — every element earns its place.
- Nautical without being kitschy — no anchors, no rope borders, no "ahoy."
- shadcn/ui defaults with Geist (Vercel's sans-serif) as the base font.
- Prioritize usability over personality. This is a tool, not a brand exercise.

## Logo
Not needed for V1. Text wordmark is sufficient.

## Priority
Ship functional by May 15. Brand polish is a post-launch concern.
